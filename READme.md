# 🐾 Cadastro de Animais API

Projeto desenvolvido para a disciplina da faculdade com o objetivo de criar uma API REST para gerenciamento de cadastro de animais.

---

# 📚 Sobre o Projeto

A aplicação permite realizar operações de cadastro de animais através de uma API REST, seguindo o padrão CRUD:

- ✅ Criar animais
- ✅ Listar animais
- ✅ Buscar animal por ID
- ✅ Atualizar informações
- ✅ Remover registros

O sistema foi desenvolvido utilizando Java com Spring Boot, aplicando conceitos de desenvolvimento back-end, arquitetura REST e persistência de dados com JPA.

---

# 🚀 Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Data REST
- Spring Validation
- Maven
- H2 Database
- Swagger / OpenAPI
